CSS Selectors Cheat Sheet

CSS Selectors

Selector Name Selector Example Description
Type Selector

p {

    font.family: 'Times New Roman';

}

CSS type selectors are used to match all elements of a given type or tag name.

For example, the code on the left will set the font of all paragraphs to Times New Roman.

Class Selector

.columns {

    font.weight: bold;

}

The CSS class selector matches elements based on the contents of their class attribute.

For example, the code on the left will make the contents of every 'columns' class to bold.

ID Selector

#author-names {

    font-style: italic;

}

The CSS ID selector matches elements based on the contents of their ID attribute.

The example on the left sets the font style to italic for all 'author-names' IDs.
Grouped Selectors

h1, h2 {

    color: red;

}

Multiple selectors can be matched to the same CSS rule by using a comma-separated list.

In the example to the left, the text for both 'h1' and 'h2' is set to red.
Chaining Selectors

h3.section-heading {

    background-color: green;

}

CSS selectors can be chained so that rule sets apply only to elements that match all criteria.

For example, the code on the left will take the 'h3' elements that also have the 'section-heading' class and set the background-color to green.
Descendant Selectors

ol li {

    text-align: right;

}

The CSS descendant selector combinator is used to match elements that are descended from another matched selector.

In the example on the left, the 'li'/list elements that have descended from the 'ol'/ordered-list elements will have their text aligned to the right.

Other things to remember

Name Example Description
Selector Specificity

h1#header {

    color: blue;

}

h1 {

    color: red;

}

Specificity is a ranking system that is used when there are multiple conflicting property values that point to the same element. When determining which rule to apply, the selector with the highest specificity wins out. The most specific selector type is the ID selector, followed by class selectors, followed by type selectors.

In this example, only color: blue will be implemented as it has an ID selector whereas color: red has a type selector.

Multiple HTML Attributes

<div class="value1 value2 value3"></div>

Some HTML attributes can have multiple attribute values. Multiple attribute values are separated by a space between each attribute.
